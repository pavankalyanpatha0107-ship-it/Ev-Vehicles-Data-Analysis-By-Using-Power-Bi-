# 🚗 Electric Vehicle Analysis Dashboard

## 📊 Project Overview

This project presents an **Electric Vehicle (EV) Analysis Dashboard** built to explore and visualize trends in EV adoption, performance, and distribution. The dashboard provides key insights into vehicle types, manufacturers, model trends, and eligibility for clean alternative fuel programs.

## 🎯 Objectives

* Analyze total electric vehicles and their distribution
* Understand average electric range across vehicles
* Explore EV adoption trends over time
* Identify top manufacturers and models
* Visualize eligibility for clean alternative fuel vehicles

## 📌 Key Features

* **Total Vehicles Overview** – Displays total registered EVs
* **Average Electric Range** – Shows overall EV efficiency
* **Interactive Filters** – Filter by City, EV Type, Utility
* **Geographical Analysis** – State-wise distribution map
* **Top Manufacturers** – Bar chart of leading brands
* **Model Insights** – Treemap of popular models
* **CAFV Eligibility** – Donut chart analysis
* **Year-wise Trends** – Growth of EVs over time

## 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel / CSV
* Power Query
